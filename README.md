# 坦克数字孪生展示

基于 Three.js + Vite 的坦克 3D 数字孪生场景，支持键盘驾驶和摄像头视觉动捕驱动。

**在线访问：** https://yjhui2005.github.io/auto_tank_digital_twin/

## 功能

- **3D 场景**：Three.js 渲染的坦克与地形数字孪生场景
- **键盘驾驶**：W/S 前进后退，A/D 转向，Q/E 调整动力上限
- **摄像头动捕**：通过摄像头识别白色鼠标（YOYO）驱动坦克移动
- **灵敏度调节**：低/中/高三档灵敏度实时切换
- **履带物理**：左右履带差速、惯性刹车、障碍碰撞检测

## 快速开始

```bash
npm install
npm run dev
```

## 部署

推送代码到 `main` 分支后，GitHub Actions 会自动构建并部署到 GitHub Pages。

## 动捕使用说明

1. 点击右下角"启用摄像头动捕"按钮
2. 允许浏览器摄像头权限
3. 将白色鼠标（YOYO）放入镜头
4. 手持鼠标移动，坦克将跟随移动方向
5. 使用灵敏度按钮切换响应速度
